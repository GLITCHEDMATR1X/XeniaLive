CodeRED Xenia RDR Bootstrap Menu v17.1
======================================

Purpose:
- Fixes the confusing behavior where CodeRED_RDR_Bootstrap_Menu_v9.bat appears to disappear or gets replaced by host/game output.
- Game launchers, bootstrap host, and AI Guest control menu now open in separate CMD windows.
- The main menu remains available for status checks, spawning AI guest, collecting logs, or opening AI controls.

Recommended flow:
1. Run CodeRED_RDR_Bootstrap_Menu_v9.bat.
2. Choose 4: Singleplayer Host - Disc 1 + AI Bridge v17.
3. Leave that game/launcher CMD alone.
4. Return to the original main menu window.
5. Choose 13 to open the AI Guest Control Menu, or 15 to spawn the AI guest quickly.

Do not start multiple SPHost game launchers at the same time.
It is okay to have these running together:
- one SPHost game launcher window
- BootstrapHost window
- AI Guest Bridge window
- AI Guest Control Menu window
- original main menu window
